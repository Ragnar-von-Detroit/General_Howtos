# start new screen session

```
screen
```

# reattach to existing screen session or list available sessions

```
screen -r
```

# kill screen session without reattaching it

```
screen -S screen_session_id -X quit
```

# detach 

```
"CRTL+A +d"
```

# kill 

```
"CRTL+A +K"
```
