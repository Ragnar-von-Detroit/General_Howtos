# seq - create and print sequence of numbers

# usage / examples

* seq <start>
* seq <start> <end>
* seq <start> <step size> <end>

## count up from 1 to x

```
#will output 1 2 3 4 5 6 7
seq 7
```

## count up from x to y

```
#will output 3 4 5
seq 3 5
```

## count up from x to y with the step size of z

```
#will output 1 4 7
seq 1 3 9
```
