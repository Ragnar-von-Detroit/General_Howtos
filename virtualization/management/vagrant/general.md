# box

## install a box

```
#base arch linux box
vagrant init terrywang/archlinux; vagrant up --provider virtualbox
#a lamp centos box
vagrant init smallhadroncollider/centos-6.5-lamp; vagrant up --provider virtualbox
```

## list available boxes

```
vagrant box list
```
